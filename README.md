# Telink Produktutbildning - AI-Assistent

En interaktiv presentation för säljteamsutbildning om Telinks AI-Assistent.

## Funktioner

- **11 kapitel** med pedagogiskt upplagd produktinformation
- **Interaktiva quiz** efter varje kapitel med poängsystem
- **Leaderboard** för teamtävling
- **Två teman** - mörkt (rekommenderat) och ljust
- **Responsiv design** - fungerar på alla enheter
- **Tangentbordsnavigering** - piltangenter + mellanslag

## Kapitel

1. Varför denna utbildning
2. Visionen
3. Kundens smärtor (prioriterat)
4. Värde - hårda & mjuka (prioriterat)
5. Kärnfunktioner (prioriterat)
6. Kunskapsbas
7. Integrationer
8. Assistent-appen
9. Implementeringsprocess (prioriterat)
10. Avtalsprocess (prioriterat)
11. Bonus: Rapporter & Machine Learning

## Installation

### Lokal utveckling
Öppna `index.html` i en webbläsare.

### Vercel Deploy
1. Pusha till GitHub
2. Anslut repot till Vercel
3. Deploy automatiskt

## Användning

1. **Starta** - Lägg till deltagarnas namn
2. **Navigera** - Piltangenter eller knappar
3. **Quiz** - Klicka "Visa svar", sedan klicka på vinnaren
4. **Leaderboard** - Uppdateras automatiskt

## Teknologi

- Vanilla HTML/CSS/JS
- Plus Jakarta Sans (Google Fonts)
- Inga externa beroenden
- Statiska filer - perfekt för Vercel
